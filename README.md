# The VKI App
All-in-one web & mobile app for VKI NGU students. With schedules, grades, absences and many other QoL features, you won't need anything else.

Big thanks to [DedMaxTech](https://github.com/DedMaxTech/VkiHub) for providing parts of essential code.

## Features
- Week schedule, with changes visibly highlighted
- Grades shown per subject, alongside the average grade for the subject
- Absences and percentage of missed lessons per subject
- Push notifications when you get a new grade, absence or the schedule changes

# The VKI App
Все-в-одном веб и мобильное приложение для студентов ВКИ НГУ. С расписанием, оценками, пропусками и множеством других полезных функций — вам больше ничего не понадобится.

- Недельное расписание с наглядным отображением изменений
